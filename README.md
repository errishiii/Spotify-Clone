# Spotify-Clone

A responsive front-end clone of the Spotify Web Player built using **HTML, CSS, and JavaScript**.  
This project focuses on recreating Spotify’s user interface, layout, and interactive behavior to strengthen front-end development skills.

---

## Project Overview

This application replicates the visual structure and user experience of Spotify’s web player, including navigation, library layout, content sections, and hover-based interactions.  
The project is **UI-focused** and does not include real music streaming, authentication, or backend services.

The main objective is to practice:
- Pixel-accurate UI development
- Responsive layouts
- Clean CSS architecture
- JavaScript-based UI interactions

---

## Features

- Spotify-inspired modern user interface
- Sticky header with navigation and search bar
- Responsive hamburger menu for smaller screens
- Sidebar library section
- Horizontally scrollable content sections:
  - Trending Songs
  - Popular Artists
  - Albums & Singles
  - Radio
  - Featured Charts
  - India’s Best playlists
- Smooth hover effects and play button animations
- Fully responsive design without layout scaling hacks
- Well-structured and readable codebase

---

## Tech Stack

- **HTML5** – Semantic markup and layout structure  
- **CSS3** – Flexbox, Grid, animations, media queries  
- **JavaScript** – UI interactivity and event handling
